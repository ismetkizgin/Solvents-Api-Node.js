Countdown Solver


# Request Example
GET countdownsolver
{
    "numbers": "2,4,5,7,5,50",
    "target": 425
}

For the countdown solver, you need to make a GET call to the URL below :
countdownsolver 



Result example:
{
    "result": "50 * 4 * 2 = 400\n5 * 5 = 25\n400 + 25 = 425\n",
    "score": 9
}

Word Solver


# Request Example
GET wordsolver
{
    "letters": "m, e, k, t, u, a,g, y"
}

For the word solver, you need to make a GET call to the URL below :
wordsolver 



Result example:
{
    "word": "MUKTEZA",
    "score": 9
}
